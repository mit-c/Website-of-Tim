Mandlebrot set zoom

Below is the iterative system behind the Mandlebrot set. \[z_0 = 0 \] \[ z_n := z_{n-1}^2+c \quad \text{for} \; z_n, \; c \, \in \mathbb{C}\] Where \(\mathbb{C}\) is the set of complex numbers and \(n\) is a natural number.
The Mandlebrot set is defined to be \[M=\{c \in \mathbb{C} : |z_n|^2 \ngeq 4\ \text{for all} \; n \in \mathbb{N}\}\] Fig.1 demonstrates the circle with radius 2 which if \(z_n\) is not inside the circle then \(z_n \to \infty\) as \(n\to\infty\)

The animation shows how many iterations (using colour) it takes to escape set: \(|z_n|^2>4\). So I set: \[\text{Colour}=\frac{n_\text{escape}}{n_\text{max}}\] I also added zooming and rotation to the animation. Zooming is tricky because you need to increase the maximum number of iterations as you zoom to retain detail. However, if you increase it too much a lot of pixels have the same colour (because more pixels are escaping). I worked out good values by trial and error but if I wanted to do a huge zoom I would.
